= Module Calculator

The idea is to identify a change within a directory tree based
on a calculated hash which is stored within a file and compared
to actual calculated hash.

== State

It's just a Proof Of Concept

== Requirements

=== Build Time
 * Apache Maven 3.8+
 * JDK11+ for building.

=== Run Time

 * JDK8+
